# Full-Stack Personal Portfolio

A professional portfolio website built with Node.js, Express, and MySQL.

## Features
- Dynamic EJS templates
- Contact form with MySQL integration (XAMPP)
- SweetAlert2 for user feedback
- Modular MVC architecture

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start XAMPP and import the `database.sql` file via phpMyAdmin.
4. Update `db/db.js` with your MySQL credentials.
5. Run `npm start` to view the project at `localhost:5000`.
